华中农业大学二手书小程序

请生成一套可直接上线运行、无 BUG、完整可用的微信小程序源码

项目名称：华中农业大学二手书交易平台

使用微信小程序原生语法 + 云开发，无需独立服务器。

一、核心定位

华中农业大学校内二手教材、教辅、考研、考证书交易平台

模式：线上担保交易 + 线下自提

平台收取 9% 社群服务费，交易完成自动扣除

主色调：绿色 + 白色（华农校色），简洁校园风

二、核心功能（必须全部实现）

1. 用户模块

微信一键登录

个人中心：我的发布、我的订单、我的收藏、消息中心

2. 书籍发布（关键）

支持 ISBN 扫码自动识别书名、作者、出版社

支持手动填写：售价、原价、成色、专业、备注

上传最多 3 张图片

卖家可随时修改价格、修改介绍、修改后依然需要审核

发布后需管理员审核

3. 搜索与分类

搜索：书名、专业、课程

分类：公共课 / 专业课 / 考研 / 考公 / 四六级 / 课外书

支持求书板块（买家求购）

4. 担保交易流程（必须严格实现）

买家下单

买家付款到平台中间账户（微信支付）

双方线下自提

买家点击确认收货

系统自动打款给卖家

自动扣除 9% 社群服务费

5. 支付与手续费

接入微信支付

9% 费率明码标价

首页弹窗公告，展示收费说明

弹窗内容支持在管理员后台随时修改，无需改代码

6. 消息通知（必须）

下单、付款、确认收货、退款提醒

服务号模板消息推送（不打开小程序也能收到）

7. 管理员后台（我使用）

书籍审核、上下架

订单管理、退款处理

交易统计、收益统计

弹窗公告后台编辑、随时更换

用户管理、违规管理

8. UI 风格

主色：绿色

背景：白色、浅灰

简洁、清爽、学生友好

三、技术要求

微信小程序原生：WXML + WXSS + JS + JSON

云开发数据库、云函数、云存储

支持微信支付、服务号消息

代码结构清晰、注释完整、可直接导入运行

四、交付内容

完整源码（按文件目录）

部署步骤教程

支付开通、服务号绑定教程

弹窗公告文案、用户须知、收费说明
